## Tentang Nuragabuana

**Nuragabuana** adalah komunitas sosial yang hadir sebagai ruang untuk belajar, berbagi, dan bertumbuh bersama.

Kami bergerak dari semangat kepedulian terhadap lingkungan, pertanian, edukasi, serta pemberdayaan masyarakat.
Bagi kami, komunitas bukan hanya tentang kegiatan besar, tetapi juga tentang langkah kecil yang dilakukan dengan konsisten dan penuh kepedulian.

> Dari langkah kecil, kami percaya perubahan baik bisa tumbuh bersama.

## Ruang Digital Nuragabuana

GitHub ini kami gunakan sebagai ruang digital untuk membangun, menyimpan, dan mengembangkan project komunitas secara lebih rapi.

Beberapa hal yang akan kami kembangkan di sini:

* Website resmi Nuragabuana
* Dokumentasi publik komunitas
* Media kit dan identitas visual
* Materi pembelajaran terbuka
* Project kolaborasi bersama relawan
* Arsip digital kegiatan dan pengembangan komunitas
* Ruang belajar dan kegiatan mengajar bersama peserta
